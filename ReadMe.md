# Yazi

## Install yazi packages

```shell
ya pack -i

ya pack -a yazi-rs/plugins:mount
ya pack -a kalidyasin/yazi-flavors:tokyonight-night
```

## yazi theme

- ayu-dark

```bash
ya pack -a kmlupreti/ayu-dark
```

```toml
[flavor]
dark = "ayu-dark"
```

- flexoki

```bash
ya pack -a gosxrgxx/flexoki-dark
```
